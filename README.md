## 📌 2. 요구사항 및 나의 역할

### 💼 개요
음식 레시피를 공유하는 **게시판(Web Board)** 프로젝트입니다.  
회원 관리, 게시글/댓글 CRUD, 추천, 검색, 페이징 등  
게시판 서비스의 **핵심 기능 전체를 구현**하는 것을 목표로 개발했습니다.

### ✔ 요구사항 요약
- 회원가입 · 로그인 · 로그아웃
- 게시글 작성 / 조회 / 수정 / 삭제(CRUD)
- 댓글 작성 / 수정 / 삭제
- 게시글 추천(좋아요) 기능
- 조회수 증가 기능
- 게시글 검색 기능
- 페이징 처리 기능

### 👨‍💻 담당 역할
- JSP/Servlet 기반 전체 웹 구조 설계 및 구현
- 회원 및 게시판 테이블 설계, ERD 모델링
- Controller-DAO-VO 기반 MVC 패턴 적용
- 페이지네이션, 검색, 추천, 조회수 등 핵심 기능 설계 및 구현

## 📌 3. 기술 스택 및 아키텍처

| 항목 | 기술 |
|------|------|
| **프레임워크** | JSP / Servlet MVC 구조 |
| **백엔드** | Java, OJDBC, DAO 패턴 |
| **프론트엔드** | HTML, CSS, JavaScript, jQuery |
| **DB** | Oracle |
| **개발 환경** | Eclipse, Tomcat 서버, GitHub |

### 📐 아키텍처 특징
- MVC 패턴 기반 레이어 분리 (Controller / DAO / VO)
- JSP를 View로 사용하여 UI와 비즈니스 로직 분리
- OJDBC를 통한 안정적 DB 연동
- 페이징·검색·조회수 증가를 위한 상태(Session) 기반 기능 적용

## 📌 5. 설치 및 실행 방법

### 1️⃣ 프로젝트 클론

git clone https://github.com/Alinate1111/BulletinboardPortfolio.git


### 2️⃣ Eclipse에서 프로젝트 Import

File → Import → Existing Projects into Workspace → Finish


### 3️⃣ Oracle DB 생성 및 SQL 파일 실행

src/main/webapp/sql_file 폴더의 SQL 스크립트를 실행하여  
member, manager, board 테이블과 시퀀스를 생성합니다.


### 4️⃣ Tomcat 서버 설정

- Servers 탭 → Tomcat 추가  
- 프로젝트를 Tomcat에 Add

### 5️⃣ 프로젝트 실행  

프로젝트 → Run As → Run on Server

### 🌐 접속 URL

Tomcat 기본 포트(8080) 기준:
👉 http://localhost:8080
